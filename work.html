Ticketing

This ticketing app was developed using Flutter and Laravel 11, designed to simplify digital ticket purchases. Users can select tickets, see the total price calculated automatically, and each transaction is instantly recorded in the purchase history. The app also supports receipt printing via Bluetooth thermal printers, making it ideal for events or on-site ticket sales.

HandCraft App UI

HandCraft Art Shop" is a Flutter-based e-commerce prototype showcasing handcrafted products. It displays high-resolution images, product details, color options, quantity settings, and an Add to Cart button. Using widgets like Column, Row, Text, Image, Icon, and ElevatedButton, the app features a simple state management system and responsive UI across different screen sizes.

Notes App

This Notes app was built using Flutter and Laravel 11, serving as a simple and responsive platform for personal note-taking. In addition to typing notes, it includes a drawing feature that allows users to freely draw directly within a note. The app also offers a user profile feature, where users can select a custom avatar as their personal identity. All notes are securely stored and accessible anytime, with a clean and intuitive interface—ideal for jotting down ideas, to-do lists, sketches, and daily journals.

MedHub

MedHub is a digital healthcare application built with Flutter and Laravel, designed to help users easily access and manage their health data. Key features include medical history tracking, doctor consultations, prescription and medication management, vaccination registration, health education articles, and emergency contacts. The app also offers doctor ratings and a direct messaging feature, supporting faster and more integrated healthcare services.

Learnify E-learning Website

Learnify is a web-based learning platform developed using Laravel, designed to help users access study materials in a structured and efficient way. Key features include access to e-books and video content, a material search system, and an interactive discussion forum where users can comment, reply, like, and rate each material. Learnify offers an engaging and user-friendly learning experience, making it ideal for modern educational needs.

Learnify Dashboard

The Learnify admin dashboard is designed to be simple and functional, allowing administrators to manage course categories, materials, and user data efficiently. Search and edit/delete actions streamline content management within the platform.

E-wallet App

The e-wallet app enables users to make digital payments such as mobile credit, electricity bills, and more, as well as top-up balances through integrated banks. Integrated with Midtrans to ensure secure and seamless transactions.

WathsApp UI

A UI clone project of WhatsApp built with Flutter to explore and study its layout structure and design system. The project includes key interface components such as the chat list view, tab navigation (Chats, Status, Calls), floating action buttons, and various interactive UI elements.

MedHub Dashboard

The MedHub dashboard provides full control to the admin for managing various aspects of the application, including users, health records, medical consultations, doctors, vaccinations, educational articles, and emergency contacts. With a summary statistics feature, the admin can monitor current activities, while user and doctor management allows for adding, editing, or deleting data. The admin can also monitor patients' health records, manage vaccination schedules, and edit health articles for education. The notification and user access settings ensure the system remains efficient and secure. Overall, this dashboard aims to improve the management of medical data and facilitate monitoring and maintaining the quality of healthcare services in MedHub.
